To Enable or Disable Audit Logging

  1. In the navigation tree, expand the Configuration node.
  2. Under the Configuration node, select the Security node.

    The Security page opens.

  3. Select the Audit Logging Enabled checkbox to enable audit logging.

    To disable audit logging, deselect the Audit Logging Enabled checkbox.

    Enabling this option causes the loading of the audit modules and ensures they are called by the Enterprise Server’s audit library at audit points.

  4. If you are enabling audit logging, specify a default audit module as described in To Set the Active Audit Module.
  5. Click Save.
  6. Restart the Enterprise Server if Restart Required displays in the console.
