New File Realm User

Use the New File Realm User page to grant access to the current realm to a new user.

The New File Realm User page contains the following options.

Realm Name

A read-only field displaying the name of the currently selected realm.

User ID

The user ID for the new user account.

Group List

Groups to which the new user will belong. Separate multiple group names with commas.

New Password

Password for the new user.

Confirm New Password

Password for the new user.
