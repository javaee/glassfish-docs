Common Tasks

The Common Tasks page provides shortcuts for common Administration Console tasks.

Use the Common Tasks page for the following operations:

  • To Register the Sun GlassFish Enterprise Server With Sun Connection

  • To View the Support Page for the Sun GlassFish Enterprise Server

  • To Deploy an Application

  • To View Monitoring Data

  • To Configure the Monitoring Service

  • To View Installed Components

  • To Update Installed Components

  • To Install Add-On Components

  • To Create a JDBC Connection Pool

The Common Tasks page also provides links to the Enterprise Server documentation.
