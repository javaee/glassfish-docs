Edit JDBC Resource

Use the Edit JDBC Resource page to modify an existing JDBC resource.

The Edit JDBC Resource contains the following options.

JNDI Name

The J.N.D.I. name is a read-only field. You can only specify the J.N.D.I. name when you create a new JDBC resource.

Pool Name

The connection pool associated with the JDBC resource.

Description

A short description of the JDBC resource.

Status

The status of the JDBC resource. The resource can be enabled or disabled.
