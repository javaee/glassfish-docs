New Network Listener

Use the New Network Listener page to create a new network listener.

The New Network Listener page contains the following options.

Name

The name of the network listener.

Status

The status of the network listener. The listener can be enabled or disabled.

This option is disabled by default.

Port

The port on which the network listener will listen.

Address

The IP address on which the network listener will listen.

Protocol

The protocol associated with the network listener.

Thread Pool

The thread pool associated with the network listener.

Transport

The type of transport for the network listener.
