Edit Audit Module

Use the Edit Audit Module page to modify an audit module.

The Edit Audit Module page contains the following options.

Name

Read-only field that displays the name of the audit module that you are editing.

Class Name

The fully qualified name of the class that implements this module. The class name for the default audit module is com.sun.enterprise.security.Audit.

Audit On

If this option is selected, the implementation class is enabled.

Additional Properties

Additional properties for the audit module. The Enterprise Server does not define any additional properties for audit modules.
