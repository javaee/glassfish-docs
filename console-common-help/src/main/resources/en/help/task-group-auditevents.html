Auditing Enterprise Server Events

Auditing how Enterprise Server records significant events, such as errors or security breaches. Enterprise Server records all authentication events in log files. A complete access log provides a sequential trail of Enterprise Server access events.

Auditing Enterprise Server Events involves the following tasks:

  • To Create an Audit Module

  • To Edit an Audit Module

  • To Delete an Audit Module

  • To Enable or Disable Audit Logging

  • To Set the Active Audit Module

  • To Use the Default Audit Module
