Manager Properties

The session manager provides the means to configure how sessions are created and destroyed, where the session state is stored, and the maximum number of sessions that are available.

Use the Manager Properties page to add or modify session manager properties.

The Manager Properties page contains the following options.

Reap Interval

The number of seconds before inactive session data is deleted from the store.

Max Sessions

The maximum number of sessions allowed.

Session File Name

The name of the file that contains session data.

Session ID Generator Class Name

The ID Generator class name that is identified in the Enterprise Server classpath.

Additional Properties

Additional properties for the session manager. The Enterprise Server does not define any additional properties for the session manager.
