Store Properties

Use the Store Properties page to add or modify store properties.

The Store Properties page contains the following options.

Reap Interval

The number of seconds before inactive session data is deleted from the store.

Directory

The directory path to the file where the session data will be stored. This is needed only if you have selected File as the persistence type.

Additional Properties

Additional properties for the store. The Enterprise Server does not define any additional properties for the store.
