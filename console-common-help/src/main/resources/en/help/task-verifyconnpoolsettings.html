To Verify Connection Pool Settings

  1. In the navigation tree, expand the Resources node.
  2. Under the Resources node, expand the JDBC node.
  3. Under the JDBC node, select the Connection Pools node.

    The Connection Pools page opens.

  4. On the Connection Pools page, click the name of the connection pool that you are verifying.

    The Edit Connection Pool page opens.

  5. On the Edit Connection Pool page, click the Ping button.

    The Administration Console attempts to connect to the database. If an error message appears, check to see if the database server was restarted.
