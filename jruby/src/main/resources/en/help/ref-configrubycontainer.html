Ruby Container

Use the Ruby Container page to configure the Ruby container.

The Ruby Container page contains the following options.

Jruby Home

The path to the directory where the JRuby container is installed. The default location is as-install/jruby. If a deployed JRuby application also specifies the JRuby Home field, the application property value overrides the value of the Jruby Home field for the container.

Runtime

The initial number of JRuby runtimes to start. The value must be equal to or greater than the value in the Minimum Runtime field but less than the value in the Maximum Runtime field. The default value is 1.

Minimum Runtime

The minimum number of JRuby runtimes in the pool. The value must be less than or equal to the value in the Runtime field and less than the value in the Maximum Runtime field. The default value is 1.

Maximum Runtime

The maximum number of JRuby runtimes in the pool. The value must be equal to or greater than the value in the Minimum Runtime field and equal to or greater than the value in the Runtime field. The default value is 2.
