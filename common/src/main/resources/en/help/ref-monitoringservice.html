Monitoring Service

Use the Monitoring Service page to configure monitoring options for individual server components or services.

For each monitoring option, available levels are LOW, HIGH, and OFF. The default level is OFF. There is no difference between levels of monitoring offered by the LOW and HIGH options in this release.

The Monitoring Service page contains the following options.

HTTP Service

HTTP service monitoring level.

Web Container

Web container monitoring level.

JVM

JVM monitoring level.
