Session Properties

Use the Session Properties page to add or modify properties for the web container session.

The Session Properties page contains the following options.

Session Timeout

Specifies the number of seconds that a session is valid. The default value is 600.

Additional Properties

Additional properties for web container sessions.

For a description of available properties, see Properties Specific to Web Container Sessions.
