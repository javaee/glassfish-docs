Domain Attributes

Use the Domain Attributes page to set advanced domain attributes for the Enterprise Server.

The Domain Attributes page contains the following options.

Application Root

The full path of the directory where applications are deployed.

Log Root

Identifies where the server log files are kept.

Locale

If this field is left blank, the default locale of the host will be used.

A locale is an identifier that specifies a particular combination of language and region. For example, the locale for American English is en_US, and for Japanese it is ja_JP. To use a non-English locale, the Application Server must be localized, a process that includes translating English into another language.
