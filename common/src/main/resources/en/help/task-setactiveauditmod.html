To Set the Active Audit Module

Before You Begin

To specify the audit module that the server uses, first enable audit logging as described in To Enable or Disable Audit Logging.

  1. In the Audit Modules field, click the name of the audit module to be used by the server.

    The Edit Audit Module page opens.

  2. Select the Audit On Enabled checkbox.
  3. Click Save.
  4. Restart the Enterprise Server if Restart Required appears in the Administration Console.
