General Information

The General Information page displays the following information.

Name

The name of the server.

HTTP Port(s)

The currently configured ports for HTTP requests and responses.

IIOP Port(s)

The currently configured ports for IIOP requests and responses.

Configuration Directory

The directory on the host machine containing configuration files, such as domain.xml.

Installed Version

The product version identifier.

Debug

Identifies whether debugging is enabled. If debugging is enabled, the port number is also displayed.

Stop Instance Button

Click the Stop Instance button to stop the application server.

In addition, you can select the following tabs to perform additional tasks:

  • JVM Settings Tab: configure the JVM general settings used by the Enterprise Server.

  • Advanced Tab: set general properties for deploying applications.

  • Monitor Tab: view monitoring data for JVM, Server, Thread Pools, HTTP Service, and Transaction Service.
