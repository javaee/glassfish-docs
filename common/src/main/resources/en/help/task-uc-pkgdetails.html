To View Detailed Information About a Component

The Update Tool provides detailed information about installed components, available updates to installed components, and add-on components.

  1. In the navigation tree, select the Update Tool node.

    The Available Add-Ons page is displayed.

  2. Click the tab for the page where the component is listed.

    The component is listed on one of the following pages:

    • Available Add-Ons

    • Available Updates

    • Installed Components

  3. In the table of components, select the component.

    The Component Details page for the selected component is displayed.
