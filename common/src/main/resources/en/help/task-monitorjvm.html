To View JVM Monitoring Data

  1. In the navigation tree, select the Application Server node.

    The General Information page opens.

  2. On the General Information page, click the Monitor tab.

    The Web Container Monitoring page opens.

  3. On the Web Container Monitoring page, click the JVM tab.

    The JVM Monitoring page opens.

  4. (Optional) From the Garbage Collectors drop-down list, select a garbage collector to view the monitoring data for that garbage collector.
  5. (Optional) Click the Refresh button to update displayed monitoring data.
