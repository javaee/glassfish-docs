To Change the Admin Password

Encrypting the admin password enhances security of your application. Consult Chapter 7, Administering System Security, in Sun GlassFish Enterprise Server v3 Administration Guide for more detailed information on security procedures.

  1. In the navigation tree, expand the Configuration node.
  2. Expand the Security node.
  3. Expand the Realms node.
  4. Select the admin-realm node.

    The Edit Realm page opens.

  5. On the Edit Realm page, click the Manage Users button.

    The File Users page opens.

  6. On the File Users page, click the user for whom you want to add or change the password.

    The Edit File Realm User page opens.

  7. In the New Password and Confirm New Password fields, type the new password.
  8. Click Save.
