To Configure the Monitoring Service

  1. In the navigation tree, expand the Configuration node.
  2. Under the Configuration node, select the Monitoring node.

    The Monitoring Service page opens.

  3. From the HTTP Service drop-down list, select LOW or HIGH to enable HTTP service monitoring.

    There is no difference between levels of monitoring offered by the LOW and HIGH options in this release. To disable HTTP service monitoring, select OFF. The default value is OFF.

  4. From the Web Container drop-down list, select LOW or HIGH to enable web container monitoring.

    There is no difference between levels of monitoring offered by the LOW and HIGH options in this release. To disable web container monitoring, select OFF. The default value is OFF.

  5. From the JVM drop-down list, select LOW or HIGH to enable JVM monitoring.

    There is no difference between levels of monitoring offered by the LOW and HIGH options in this release. To disable JVM monitoring, select OFF. The default value is OFF.

  6. Click Save.
